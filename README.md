# Simple Contact Manager

A full-stack web application that allows users to securely manage personal contacts.  
Users can register, log in, and create, search, update, and delete contacts.

## Features
- User authentication with hashed passwords
- Secure login and registration system
- Create, view, update, and delete contacts
- Search contacts dynamically
- Frontend and backend communication using REST APIs

## Technologies Used
Frontend:
- HTML
- CSS
- JavaScript
- AJAX

Backend:
- PHP
- REST API

Database:
- MySQL

Deployment:
- Apache Server
- DigitalOcean

## How It Works
1. Users create an account and log in securely.
2. The frontend sends requests to the backend REST API.
3. The backend processes requests and interacts with the MySQL database.
4. Contact data is returned as JSON and dynamically displayed on the frontend.

## Project Structure
frontend/   → HTML, CSS, and JavaScript user interface
backend/    → PHP REST API and server logic
database/   → MySQL schema and database setup
